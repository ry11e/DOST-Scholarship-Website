# Scholarship Management System
A web application to digitize and manage scholarship applicants.

## 🛠 Tech Stack
- **Backend:** PHP 8.x
- **Frontend:** Bootstrap, Google Fonts, ApexCharts, Bootstrap4.DataTables 
- **Database:** MySQL (MariaDB)
- **Charts:** ApexCharts

## 🚀 Getting Started
1. **Database:** Import `scholarship_db.sql` into your phpMyAdmin.
2. **Config:** Update `db_connect.php` with your database credentials.
3. **Run:** Access the project via `localhost/scholarship_project/admin`.

## 📁 Key Folders
- `/vendors`: External CSS/JS libraries.
- `/src/plugins`: DataTables and chart plugins.
- `/includes`: Reusable PHP components like `head.php`, `connection.php`, and `sidebar.php`.
