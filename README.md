# Scholarship Management System
A web application to digitize and manage scholarship applicants. Initial Goal was to house the records for   DOST Scholars in AKlan

## 🛠 Tech Stack
- **Backend:** PHP 8.x
- **Frontend:** Bootstrap, Google Fonts, ApexCharts, Bootstrap4.DataTables 
- **Database:** MySQL (MariaDB)
- **Charts:** ApexCharts
- **Server** Xampp for local development


## 🚀 Getting Started
1. **Database:** Import `scholarship_db.sql` into your phpMyAdmin.
2. **Config:** Update `includes/connection.php` with your database credentials.
3. **Directory** Move/Copy project folder into your `xampp/htdocs` folder.
4. **Run:** Access the project folder, e.g `localhost/scholar`.

## 📁 Key Folders
- `/vendors`: External CSS/JS libraries.
- `/src/plugins`: DataTables and chart plugins.
- `/includes`: Reusable PHP components like `head.php`, `connection.php`, and `sidebar.php`.
- `/uploads`: Houses uploaded files
